# Workflows

This folder contains workflow definitions for the Florida Ramp & Lift Operational Intelligence Platform.

Workflow docs should define trigger, inputs, outputs, review gates, exception states, and what remains manual.

## Planning Docs

- `dispatch-workflow.md` defines the approved-job-to-dispatch flow.
- `dispatch-workflow.md` defines dispatch-ready assignments and transfer review for partial, incomplete, or return-needed work.
- `job-marketplace-and-assignment-engine.md` defines the controlled job pool, claim/bid, smart-offer, assignment workflow, and continuation recommendations for transfer work.
