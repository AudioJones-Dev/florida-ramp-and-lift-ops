# TOOL AND PPE CHECKLIST

**Audience:** Every crew member, every day, before leaving the yard.
**Format:** Print this. Tape it inside the truck. Reference before the first job of every day.

---

## 1. Mandatory PPE (every job, every day, no exceptions)

- [ ] **Steel-toed boots.** Not work boots. Steel-toed.
- [ ] **ANSI Z87+ safety glasses.** Actual safety glasses. Not sunglasses. Not corrective lenses without a Z87+ marking.
- [ ] **Gloves.** Cut-resistant or work gloves appropriate for materials being handled.

**If you don't have all three, you do not start the work.** No exceptions.

---

## 2. PPE per job context

Add these based on the job:

### Working at height (any stairs / decks / scaffolding)

- [ ] Hard hat
- [ ] Fall protection (harness + lanyard) if above 6 ft

### Roadside / active equipment yard / construction site

- [ ] Hi-vis vest

### Power tool operation > 1 hour sustained

- [ ] Hearing protection (foam plugs or earmuffs)

### Cutting / grinding / dust-generating work

- [ ] N95 or respirator
- [ ] Additional eye + face protection if grinding (face shield over Z87+)

### Electrical work (VPL, lift install, anything with line voltage)

- [ ] Insulated gloves
- [ ] Lockout-tagout kit + tags (verify power is off)

### Hot weather (Florida default for most of the year)

- [ ] Water in the truck (always)
- [ ] Long sleeves or sunscreen
- [ ] Hat

---

## 3. Personal tool kit (carry every day)

The crew member's responsibility:

- [ ] Tape measure (25 ft minimum)
- [ ] Speed square / combination square
- [ ] Pencil + marking tool
- [ ] Utility knife (sharp blade)
- [ ] Levels (4-ft + torpedo)
- [ ] Hex / Allen key set
- [ ] Adjustable wrench
- [ ] Drill / driver (with batteries + chargers)
- [ ] Impact driver
- [ ] Bits (hex, Phillips, Torx common sizes)
- [ ] Hammer
- [ ] Cordless circular saw (if assigned by lead)
- [ ] Flashlight / headlamp
- [ ] Multimeter (electrical work)

---

## 4. Job-specific tools (provided by Florida Ramp)

The lead confirms these are loaded before leaving the yard:

### Ramp install / modification / recovery

- [ ] Ramp kit components for the work order
- [ ] Switchback hardware (if applicable)
- [ ] Handrail components
- [ ] Aluminum deck panels (size per work order)
- [ ] Fasteners (correct sizes for the materials)
- [ ] Ground stakes / anchors (for soft ground)

### VPL install

- [ ] VPL unit + install kit
- [ ] Electrical kit (wire, conduit, fittings)
- [ ] Mounting hardware specific to the unit
- [ ] Lockout-tagout kit

### Stair lift install

- [ ] Rail components (straight or curved per work order)
- [ ] Stair lift unit
- [ ] Mounting hardware (specific to stair type)
- [ ] Electrical kit if line-voltage powered

### Knockdown / return haul

- [ ] Load-securing equipment (straps, blankets)
- [ ] Disassembly tools per the original install type
- [ ] Inventory checklist for components being returned

### Plan-set deliverables

- [ ] Plans printed or accessible (per work order)

---

## 5. Truck checklist (before leaving the yard)

The lead confirms:

- [ ] All job-specific tools loaded (per § 4)
- [ ] All materials loaded
- [ ] Personal tool kits accounted for
- [ ] Water + hydration supplies
- [ ] First aid kit on board
- [ ] Vehicle fueled / charged
- [ ] Dispatch summary reviewed
- [ ] Customer contact info confirmed (phone + email per work order)

---

## 6. End-of-day truck checklist

Before leaving the last job:

- [ ] All tools accounted for and stowed
- [ ] All leftover hardware secured
- [ ] All trash removed
- [ ] Truck secured

Back at the yard:

- [ ] Batteries on chargers
- [ ] Any damaged tools flagged for repair / replacement
- [ ] Tomorrow's dispatch reviewed

---

## 7. Tool inspection (weekly)

The lead inspects tools and:

- [ ] Confirms power tools operate correctly
- [ ] Replaces damaged drill bits / driver bits
- [ ] Confirms saw blades are sharp / replaces dull blades
- [ ] Replaces frayed cords
- [ ] Confirms PPE supplies are stocked (gloves, safety glasses inventory, N95 stock)

---

## 8. Replacement / damage reporting

If a tool is broken, lost, or stolen:

- [ ] Report to admin same day
- [ ] Document with photo + brief description
- [ ] Admin processes replacement

If PPE is damaged or doesn't fit:

- [ ] Report to admin same day
- [ ] Get a replacement before the next job

**Never run a job with damaged or missing PPE.** It's not negotiable.

---
